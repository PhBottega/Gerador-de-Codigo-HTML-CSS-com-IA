🚀 Gerador de Código HTML & CSS com IA

Este projeto foi desenvolvido durante o aulão Missão Programação Do Zero com IA.

💡 Sobre o projeto

Uma aplicação web que utiliza Inteligência Artificial para gerar código HTML e CSS com base no que o usuário digita.

O sistema gera o código e também exibe o resultado visual em tempo real.

⚙️ Tecnologias utilizadas
HTML
CSS
JavaScript
Node.js
Express
API de IA (Groq)
Llama-3.3-70b-versatile
🔐 Segurança

A API Key não é exposta no front-end.
Foi implementado um backend em Node.js para proteger as credenciais, seguindo boas práticas.

🧠 Como funciona
O usuário digita o que deseja criar
O front-end envia a requisição para o backend
O backend chama a API de IA
O resultado é retornado e exibido na tela

🚀 Como rodar o projeto
1. Clone o repositório
git clone https://github.com/PhBottega/Gerador-de-Codigo-HTML-CSS-com-IA
2. Instale as dependências
npm install
3. Crie um arquivo .env
API_KEY=sua_chave_aqui
4. Rode o servidor
node server.js
5. Abra o front-end normalmente
